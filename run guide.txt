Automated AI Run Guide

Use these short commands from the project root.

Web:
.\scripts\Run-StarTrackAI.ps1 web

Android:
.\scripts\Run-StarTrackAI.ps1 android

What is automatic:
- local proxy starts in the background
- Open API.txt is loaded by the proxy
- Flutter receives one proxy base URL only
- Android gets adb reverse, so network/IP changes do not matter

AI coverage:
- recommendations: openAiRerank
- skill patterns: openAiSkillPatterns
- chatbot: openAiChatbot
- project approval: openAiProjectValidation

Stop proxy:
.\scripts\Run-StarTrackAI.ps1 stop

Release APK with Cloud AI:
.\scripts\Run-StarTrackAI.ps1 release

Deploy Cloud AI functions:
.\scripts\Run-StarTrackAI.ps1 deploy-cloud
