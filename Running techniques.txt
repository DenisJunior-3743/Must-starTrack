What this does automatically:
- starts the local OpenAI proxy if it is not already running
- loads Open API.txt into the proxy, not into the Flutter app
- uses one URL for all AI features: http://127.0.0.1:8787
- enables adb reverse for Android, so Wi-Fi/IP changes do not break local AI
- powers recommendations, chatbot, skill patterns, and project approval

Release mode uses Cloud Functions instead of the local laptop proxy.
