MUST STARTRACK - SIMPLE LOCAL AI RUN COMMANDS

One proxy powers everything locally:
- recommendations
- skill patterns
- chatbot
- project approval

The app now uses one Dart define:
OPENAI_PROXY_BASE_URL=http://127.0.0.1:8787

Android also uses 127.0.0.1 because the runner enables adb reverse automatically:
phone/emulator localhost:8787 -> laptop localhost:8787.
That means Wi-Fi/network changes do not matter for normal debug runs.

======================================================================
RUN WEB WITH AI
======================================================================
.\scripts\Run-StarTrackAI.ps1 web

======================================================================
RUN ANDROID WITH AI
======================================================================
.\scripts\Run-StarTrackAI.ps1 android

Optional specific device:
.\scripts\Run-StarTrackAI.ps1 android -Device RFGYC1PEY6

======================================================================
ASSEMBLE DEBUG APK WITH LOCAL AI
======================================================================
.\scripts\Run-StarTrackAI.ps1 apk-debug

Output:
build\app\outputs\flutter-apk\app-debug.apk

This is a debug APK and still uses:
OPENAI_PROXY_BASE_URL=http://127.0.0.1:8787

Keep USB debugging connected so adb reverse keeps Android localhost:8787
mapped to your laptop localhost:8787.

======================================================================
START ONLY THE AI PROXY
======================================================================
.\scripts\Run-StarTrackAI.ps1 proxy

Health check:
Invoke-RestMethod http://127.0.0.1:8787/health

======================================================================
STOP THE AI PROXY
======================================================================
.\scripts\Run-StarTrackAI.ps1 stop

======================================================================
LOCAL RELEASE APK TEST (USB + ADB REVERSE)
======================================================================
.\scripts\Run-StarTrackAI.ps1 apk-local

Note: this builds a release APK with the local proxy URL. It still needs
USB debugging with adb reverse active while testing.

======================================================================
REAL RELEASE APK (CLOUD AI)
======================================================================
Deploy functions once:
.\scripts\Run-StarTrackAI.ps1 deploy-cloud

Build release:
.\scripts\Run-StarTrackAI.ps1 release

This release APK works away from your laptop because it uses Cloud Functions.
