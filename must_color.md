# 🧠 3. How to use these in your mobile app (BEST PRACTICE)

## ✅ A. Navigation & App Identity

* It’s what users will recognize instantly as MUST

---

## ✅ B. Main Content Area

Use:

* ⚪ White + light gray

```css
background-color: #F2F2F2;
```

👉 Keep it clean — don’t overload with color

---

## ✅ C. Buttons & Actions

Primary buttons:

```css

color: white;
```

Secondary / highlights:

```css
background-color: #F4B400; /* Gold */
```

---

## ✅ D. Status Colors (VERY SMART USE)

Use colors meaningfully:

* 🟢 Green → success / active
* 🔴 Red → errors / delete
* 🟡 Gold → warning / highlight
* 🔵 Blue → primary actions

---

## ✅ E. Cards & Sections

👉 Use white cards on gray background (like the quote section)

# 🔥 4. Pro UI Advice (THIS IS WHAT MAKES IT CLEAN)

## ❌ Don’t do this:

* Don’t mix green + blue + gold everywhere randomly
* Don’t use bright colors for text

---

## ✅ Do this instead:

### Rule: **60 – 30 – 10**

* 60% → White / gray (background)
* 30% → Green / Blue (structure)
* 10% → Gold (attention)

---

## 💡 Example Layout (Mobile App)

* Top bar → 🟢 Green

* Cards → ⚪ White
* Buttons → 🔵 Blue

# 🚀 5. If you want it to look PROFESSIONAL

Add this to your design thinking:

* Use **green sparingly** → branding only
* Use **blue for actions** → consistent UX
* Use **gold only to draw attention**
* Keep spacing and padding clean
